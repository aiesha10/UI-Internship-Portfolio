# DevConnect UI/UX Design Assessment

## Project Overview
A modern, futuristic platform connecting developers and users. Designed for TrustNexus internship assessment.

## Screens Included
- Landing Page
- Role Selection (Developer/User)
- Login Screen
- Signup Screen
- Developer Dashboard

## Design System
- **Color Palette:** Dark blue `#202F5D`, Neon cyan `#00D4FF`, Purple `#A855F7`
- **Effects:** Glassmorphism, background blur, neon glows
- **Typography:** Clean sans-serif, bold hierarchy
